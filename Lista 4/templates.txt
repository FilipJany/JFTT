-Equal
	//pomysl: sprawdzic czy (a - b = 0)
	//zalozenia:
		a w 0
		b w 1
	//zasada dzialania:
		jezeli a = b to w rej 'a' zostawimy 1 ////rej 'a' to ten specjalny rejestr
		jezeli a != b to w rej 'a' zostawimy 0
		
	//linia 0: 	LOAD 0	//wczytaj a
				SUB 1	//a - b
				JG 7	//jezeli wwynik odejmowania jest != 0 to skocz do linii 7 i wpisz 0 do rej 0 (nie sa rowne)
				LOAD 1
				SUB 0
				JG 7
				INC		//byly rowne zostawiam 1 w 'a'
				
	//linia 7:	ZERO

-Less than:
	w sumie tez prawie tak samo jak equal
	w rejestrze 0 mamy a
	w rejestrze 1 mamy b
	jezeli a < b to a rej specjalnym 'a' zostawmy 1 a w przeciwnym wypadku 0
	
	//linia 0:	LOAD 0	//zaladuj a
				SUB 1	//wykonaj a - b
				JG 4	//jezeli wynik > 0 to znaczy ze a jest wieksze wiec skocz do linii 4 i zostaw 0
				INC		//a bylo mniejsze niz 0, czyli wpisz do spec rej 'a' 1
				
	//linia 4:	ZERO	//wpisz do spec rej 'a' 0 - a nie jest mniejsze niz b
